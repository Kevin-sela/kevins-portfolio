# Kelvin Ofori — Portfolio

Interactive, recruiter-friendly portfolio built with **Next.js + React + TypeScript + Tailwind CSS**.

## Quick start

```bash
npm install
npm run dev
```

## Customize content

- Edit content in `src/content/site.ts`
- Replace resume at `public/resume.pdf`
- Replace Open Graph image at `public/og.svg`
- Replace project images in `public/projects/` (drop in real screenshots anytime)

## Libraries used

- Next.js, React, TypeScript
- Tailwind CSS (+ `@tailwindcss/line-clamp`)
- Framer Motion (animations)
- TanStack React Query (GitHub widget caching)
- React Hook Form + Zod (validated contact form)
- lucide-react (icons)
